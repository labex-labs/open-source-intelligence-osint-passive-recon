OSINT 技術を活用したパッシブ偵察（受動的偵察）を実施します。ドメイン列挙、DNS インテリジェンス収集、公開データの収集手法を習得し、組織の外部攻撃対象領域（アタックサーフェス）をマッピングする方法を学びます。
